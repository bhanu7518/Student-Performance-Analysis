# Student-Performance-Analysis
Student Performance Analysis using Python, Pandas, Matplotlib, Seaborn, and Random Forest Regression. The project includes data cleaning, exploratory data analysis, visualization, correlation analysis, and machine learning-based grade prediction.
# Student Performance Analysis

This project analyzes student academic performance using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data Cleaning
- Exploratory Data Analysis
- Correlation Heatmap
- Failure Analysis
- Parent Education Analysis
- Random Forest Grade Prediction

## Results
- R² Score: 0.847
- MAE: 1.11

## Dataset
Student Performance Dataset (student-mat.csv)
